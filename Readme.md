# 🤖 Multi-Agent AI Chatbot

A simple AI chatbot that integrates **multiple AI agents into a single interface**.  
The system routes user queries to specialized agents and returns responses through a unified chatbot UI.

## 🚀 Features

- 5 AI Agents
- Router Agent for query routing
- Single chatbot interface
- Groq API integration
- Node.js + Express backend
- Futuristic UI

## 🧠 AI Agents

- Chat Agent – general conversation  
- Code Agent – programming help  
- Math Agent – solves math problems  
- Summary Agent – summarizes text  
- Research Agent – detailed explanations  

## ⚙️ How It Works

User Query → Router Agent → Selected AI Agent → Groq API → Response in Chat UI
